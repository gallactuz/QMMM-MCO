QM/MM Data for Multicopper Oxidases Study

This repository contains all QM/MM data associated with the manuscript:

"Redox–Proton Coupling Reveals a Universal Oxidation Bottleneck in Multicopper Oxidases: A QM/MM Study"

The data include:

QM/MM optimized structures for all catalytic intermediates and charge states
ORCA output files generated from QM/MM calculations
Input files used for geometry optimizations and single-point energy calculations

System description:

The QM region comprises the trinuclear copper cluster (TNC), including the three copper ions, coordinating histidine residues, the acidic residues E463 and D105, bound oxygen intermediates, and nearby water molecules. The remaining system was treated at the molecular mechanics (MM) level.

Calculations were performed using the AMBER package (sander module) interfaced with ORCA, employing an electrostatic embedding QM/MM scheme.

Directory organization:

Each folder corresponds to a specific intermediate (O2, OOH, HOOH, H2O-related species) and contains the corresponding input and output files for different charge states.

Notes:

These data are provided to ensure reproducibility and to allow direct inspection of the electronic structure calculations reported in the manuscript.

For further information, please refer to the manuscript or contact the authors.
